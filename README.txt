Projet n°2 - Réservia par Sonia Appiah



Bonjour et bienvenue ! 

Voici ma proposition pour la maquette du site de planification de vacances, Réservia.

Le développement du site a été fait en partant des éléments fournis par l'équipe, à savoir : 

	- la maquette pour la résolution ordinateur et la maquette pour la résolution mobile ;
	- les images des hébergements ;
	- le logo Réservia ;
	- les codes couleurs de la charte graphique ;
	- la police d'écriture, Raleway ;
	- la bibliothèque FontAwesome.


Les conseils et consignes : 

	- le pas utiliser de framework ou de pré-compilateur ;
	- Visual Studio Code recommandé avec ses plugins qui s'avèrent très utiles ;
	- utiliser la bibliothèque FontAwesome qui est riche en icônes et fonctionnalités ;
	- privilégier Flexbox mais Grid peut être utilisé aussi ;
	- utiliser les balises sémantiques
	- le code ne doit avoir aucune erreur ni alerte au validateur W3C HTML et CSS ;
	- site compatible avec les dernières versions de Chrome et Firefox ;
	- séparer le HTML du CSS ; 
	- versionner le code avec Git.


Notes concernant la réalisation du projet : 

	- l'intégration de la maquette a été faite sur Visual Studio Code ;
	- la bibliothèque FontAwesome a été exploitée ;
	- Sass et Bootstrap n'ont pas été utilisés.
